Flash Chat App

A real-time group chat application enabling instant messaging using Firebase.

Project Overview

Flash Chat is a modern messaging application that allows users to sign up, log in, and communicate in a shared chat room in real-time. The project's purpose was to demonstrate a full-stack mobile application, handling everything from user authentication to a live database connection for instant message updates. The interface is clean, featuring message bubbles and a smooth, auto-scrolling message list.

Challenges & Solutions

The core challenge was implementing a real-time, bidirectional data flow for messages. A traditional REST API would be inefficient. I overcame this by integrating Firebase Firestore, which provides real-time listeners. I used a `StreamBuilder` widget in Flutter to subscribe to the messages collection in Firestore. This widget automatically listens for changes and rebuilds the UI instantly whenever a new message is added to the database, creating a seamless, real-time chat experience for all users.

Tech Stack

Flutter Firebase Firestore Firebase Auth StreamBuilder
