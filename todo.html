Todo App

A productivity app to manage daily tasks with categories, priorities, and persistence.

Project Overview

The Todo App is a simple yet powerful tool to help users organize their daily lives. Users can add tasks, mark them as complete, delete them, and assign priority levels. The core feature is its ability to remember tasks even after the app is closed, ensuring users never lose their important to-do lists. The UI is designed to be clean, minimal, and fast.

Challenges & Solutions

The main technical hurdle was implementing local data persistence efficiently. Storing a list of custom task objects required a strategy beyond simple key-value pairs. I solved this by using the `shared_preferences` package. I created methods to encode the list of task objects into a JSON string before saving, and to decode it back into a Dart list upon loading. This provided a lightweight and effective way to persist complex data locally without the overhead of a full database.
